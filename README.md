# kakao-pay-test
워크플랫폼팀 개발자 지원 사전과제 전형
## 과제개요
고객이 문의하면 상담사가 접수-답변할 수 있는 응대-문의 시스템 구현
## 기술구성
    * Back-End
      Spring-Boot 2.6.8
      Spring-Security
      JPA(Hibernate)
      JAVA 11
    
    * DB
      H2 (In-Memory DB)
    
    * Front-End
      react 18.1.0
      react-router-dom 6.3.0
